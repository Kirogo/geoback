## 1. System Overview

**GeoBuild** is a comprehensive project monitoring and disbursement tracking system for NCBA Bank. It digitizes the manual process of site visit reporting, geo-tagging photos, and facilitating communication between Relationship Managers (RMs) and Quantity Surveyors (QSs) for construction loan disbursements.

### Core Business Flow

```
Customer Request â†’ RM Site Visit â†’ QS Review â†’ Approval â†’ Disbursement
```

### Key Features

| Feature | Description |
|---------|-------------|
| **Geo-tagged Photos** | Photos captured during site visits are automatically stamped with GPS coordinates to prevent fraud |
| **IBPS Integration** | Links to bank's core system for facility validation |
| **Role-based Access** | Separate dashboards for RMs and QSs |
| **Task Locking** | Prevents multiple QSs from reviewing the same report |
| **Audit Trail** | Complete history of all actions and comments |
| **PDF Generation** | Downloadable approval reports with bank branding |

### Base URL
```

```

### Authentication Endpoints

| Method | Endpoint | Description | Request Body | Response |
|--------|----------|-------------|--------------|----------|
| **POST** | `/auth/register` | Register new user | `{email, firstName, lastName, password, role?}` | `{token, user}` |
| **POST** | `/auth/login` | Login user | `{email, password}` | `{token, user}` |
| **GET** | `/auth/me` | Get current user | - | `{user}` |

### Facility Endpoints

| Method | Endpoint | Description | Parameters | Response |
|--------|----------|-------------|------------|----------|
| **GET** | `/facility/{ibpsNumber}` | Get facility details | `ibpsNumber` | `FacilityDto` |
| **GET** | `/facility/{ibpsNumber}/validate` | Validate IBPS | `ibpsNumber` | `boolean` |

| Method | Endpoint | Description | Request Body | Response |
|--------|----------|-------------|--------------|----------|

| **GET** | `/SiteVisitReport/pending` | Get pending reports | - | `List<SiteVisitReportResponseDto>` |
| **POST** | `/SiteVisitReport/{id}/lock` | Lock for review | `LockReportDto` | `{message}` |
| **POST** | `/SiteVisitReport/{id}/comments` | Add comment | `ReportCommentDto` | `{message}` |
| **POST** | `/SiteVisitReport/{id}/return` | Return to RM | `QSActionDto` | `{message}` |
| **POST** | `/SiteVisitReport/{id}/approve` | Approve report | `QSActionDto` | `{message}` |
